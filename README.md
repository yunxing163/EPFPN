# EPFPN for Object Detection

EPFPN（Enhanced Path Feature Pyramid Network）是一个用于通用目标检测任务的特征增强网络框架，兼容多种主流检测器架构，旨在提升目标检测精度与特征融合效果。

## 🚀 特性 Features

- ⚡ 支持主流目标检测器（如 Faster R-CNN、RetinaNet 等）
- 🔁 更优的特征融合路径，提升多尺度检测能力
- 🔬 支持灵活集成到不同主干网络（如 ResNet、Swin 等）
- 📈 提升小目标与复杂场景下的检测表现
